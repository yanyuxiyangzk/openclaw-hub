## 二、系统架构

### 2.1 整体架构

```
OpenClawHub SaaS (Vue3 + FastAPI)
         ↓ API / WebSocket
OpenClawHub Backend (FastAPI)
         ↓ HTTP / subprocess
OpenClawHarness (本地执行引擎)
         ↓
Claude Code / OpenCode
```

### 2.2 OpenClawHarness对接

| 方式 | 地址 | 说明 |
|------|------|------|
| HTTP API（推荐）| http://localhost:8001 | POST /execute |
| subprocess | 直接调用脚本 | 备选方案 |

**超时：** 300秒 | **重试：** 3次
